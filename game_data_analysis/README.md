# 游戏数据分析项目

## 项目概述
本项目基于游戏运营数据进行分析，生成详细的数据分析报告和可视化图表。

## 文件结构
```
game_data_analysis/
├── README.md                    # 项目说明文件
├── 游戏数据分析报告.md          # 详细的数据分析报告
├── generate_analysis_charts.py  # 图表生成脚本
└── *.png                       # 生成的分析图表
```

## 数据来源
- 数据文件：`d:/cursor/测试数据.csv`
- 数据包含：582条游戏记录
- 字段：类型、游戏、用户数量、投注金额、游戏轮数、支出金额、净盈利

## 主要发现

### 游戏类型表现
- **Lottery (彩票类)**: 用户基数最大，盈利稳定
- **Crash (崩盘类)**: 高投注金额，高盈利潜力
- **Slot (老虎机类)**: 单次投注金额大，盈利能力强
- **Arcade (街机类)**: 用户参与度高，但部分游戏存在亏损

### 关键指标
- 整体平台盈利良好
- Fortune Gems (Slot类) 是最大的盈利游戏
- Go Labubu! (Lottery类) 拥有最多的用户
- 需要关注亏损游戏的优化
