## 🚨 **关键错误记录**

### 1. **数据过滤错误**
**问题**: 没有正确过滤test01测试账户和非GP厂商数据
**症状**: 统计包含测试数据，金额不准确
**解决方案**:
```python
# 正确的过滤逻辑
if merchant_name == "test01":
    continue
# 注意：实际数据中可能没有厂商字段，需要根据实际情况调整
```

### 2. **账单生成方式错误**
**问题**: 按单个账户生成账单，而不是按主商户生成
**症状**: 生成57个单独账单而不是19个主商户账单
**解决方案**:
- 使用`matched_merchant_excel_data_*.json`文件
- 按主商户分组，每个主商户包含多个子账户
- 文件名格式：`主商户名_2025年08月_主商户账单.pdf`

### 3. **数据源选择错误**
**问题**: 使用错误的数据文件进行计算
**错误数据源**: `adapted_billing_data_2025_08.json`
**正确数据源**: `matched_merchant_excel_data_20250901_101424.json`
**原因**: Excel数据包含已计算的费用和汇率信息

### 4. **PDF生成器方法名错误**


### 8. **费率使用错误**
**问题**: 硬编码费率为5.0%，而不是使用统一商户表中的实际费率
**症状**: 所有商户显示相同费率，不符合实际情况
**数据源错误**: 
- ❌ `matched_merchant_excel_data.json` 中的 `fee` 字段大多为0.0
- ✅ `unified_merchant_table.json` 中的 `fee_rate` 字段包含正确费率

**解决方案**:
```python
# 错误做法
pdf_data = {
    'fee_rate': 5.0,  # ❌ 硬编码费率
}

# 正确做法
# 1. 加载统一商户表
with open('unified_merchant_table.json', 'r', encoding='utf-8') as f:
    unified_list = json.load(f)
    unified_merchant_data = {m['merchant_name']: m for m in unified_list}

# 2. 获取商户实际费率
if merchant_name in unified_merchant_data:
    fee_rate = unified_merchant_data[merchant_name].get('fee_rate', 5.0)
else:
    fee_rate = 5.0  # 默认费率

pdf_data = {
    'fee_rate': fee_rate,  # ✅ 使用商户实际费率
}
```

**费率分布**: 0%, 2.5%, 3.0%, 5.0%, 6.0%, 8.0%, 10.0%等不同费率
**注意**: 必须从 `unified_merchant_table.json` 获取费率，不是从Excel数据

### 9. **变量引用错误**
**问题**: 在函数中使用了未定义的变量名
**症状**: `"main_merchant_name" 未定义` 错误
**常见错误**:
```python

    fee_rate = self.get_merchant_fee_rate(main_merchant_name)  # ❌ 变量名错误
```
**正确做法**:
```python
# ✅ 正确：使用正确的变量名
def generate_pdf_for_main_merchant(self, merchant_data):
    merchant_name = merchant_data['merchant_name']
    fee_rate = self.get_merchant_fee_rate(merchant_name)  # ✅ 使用正确变量名
```

```python
def load_data(self):
    # 使用正确的数据文件
    with open('matched_merchant_excel_data_20250901_101424.json', 'r', encoding='utf-8') as f:
```

```python
def prepare_pdf_data(self, main_merchant_name, sub_accounts):
    pdf_data = {
        'merchant': main_merchant_name,      # 使用'merchant'
        'period': '2025年08月',
        'fee_rate': 5.0,
        'details': [],                       # 使用'details'
        'total_amount': total_charge_usd
    }
    
    # 为每个子账户和货币创建条目
    for account_info in sub_accounts:
        for currency_data in account_info.get('currencies', []):
            detail_entry = {
                'account': account_info.get('D_account', ''),
                'currency': currency_data.get('currency', 'USD'),
                'total_bet': currency_data.get('total_bet', 0),
                'total_prize': currency_data.get('total_prize', 0),
                'total_win_lose': currency_data.get('net_win', 0),
                'exchange_rate': currency_data.get('usd_rate', 1.0),
                'fee_rate': 5.0,
                'charge_usdt': currency_data.get('charge_usdt', 0)
            }
            pdf_data['details'].append(detail_entry)
    
    return pdf_data
```
