### 2. n8n 工作流集成（里程碑 2）

#### Webhook API
创建了 4 个 API 接口：

| 接口 | 方法 | 功能 | 文件 |
|------|------|------|------|
| `/api/webhook/query/sql` | POST | 执行 SQL 查询 | `backend/routes/webhook.js` |
| `/api/webhook/query/natural` | POST | 自然语言查询 | `backend/routes/webhook.js` |
| `/api/webhook/query/quick` | GET | 快速查询 | `backend/routes/webhook.js` |
| `/api/webhook/health` | GET | 健康检查 | `backend/routes/webhook.js` |

#### 认证系统
- ✅ API Key 认证中间件 (`backend/middleware/apiKeyAuth.js`)
- ✅ 支持 Header 和 Query Parameter 两种认证方式
- ✅ 多密钥配置和管理

#### 测试工具
- ✅ 自动化测试脚本 (`backend/test-webhook-api.js`)
- ✅ Athena 连接测试 (`backend/test-athena-connection.js`)
- ✅ Postman 测试集合 (`Athena-Webhook-API.postman_collection.json`)

### 3. n8n 工作流配置

创建了 3 个可直接导入的工作流：

#### 工作流 1: 智能查询
**文件**: `n8n-workflows/1-intelligent-query.json`

**功能**:
- 接收自然语言查询
- 自动转换为 SQL
- 执行 Athena 查询
- 生成 Excel 导出
- 支持 Lark/Web 多渠道返回

**节点数**: 9 个
**预计执行时间**: 10-30 秒

#### 工作流 2: 定时查询报告
**文件**: `n8n-workflows/2-scheduled-reports.json`

**功能**:
- 定时触发（默认每天 9:00）
- 自动查询多个指标
- 汇总生成 Excel 报表
- 发送到 Lark 邮箱
- 发送 Lark 群通知

**节点数**: 11 个
**预计执行时间**: 1-3 分钟

#### 工作流 3: 游戏评级报告
**文件**: `n8n-workflows/3-game-rating-report.json`

**功能**:
- 输入游戏名称
- 查询多维度数据（收入/活跃/付费/留存）
- 按规则计算评级
- 生成 Lark 交互卡片
- 生成 Excel 详细报告

**节点数**: 14 个
**预计执行时间**: 20-40 秒

## 🎯 核心功能实现

### 功能 1: 智能查询

**实现路径**:
```
用户输入自然语言
  ↓
n8n Webhook 接收
  ↓
调用 /api/webhook/query/natural
  ↓
Athena Service 执行查询
  ↓
返回结果 + 生成 Excel
  ↓
发送到用户
```

**技术亮点**:
- ✅ 自然语言理解（通过 Python 代码生成服务）
- ✅ 自动 SQL 生成
- ✅ 查询优化和成本控制
- ✅ 多格式导出

### 功能 2: 定时查询报告

**实现路径**:
```
n8n Schedule Trigger
  ↓
构建多个 SQL 查询
  ↓
并行执行查询
  ↓
汇总结果
  ↓
生成 Excel 报表
  ↓
发送邮件 + Lark 通知
```

**技术亮点**:
- ✅ Cron 定时调度
- ✅ 多查询并行执行
- ✅ 智能结果汇总
- ✅ 多渠道推送

### 功能 3: 游戏评级报告

**实现路径**:
```
输入游戏名称
  ↓
构建 4 个维度查询
  ↓
执行查询获取数据
  ↓
按评级规则计算分数
  ↓
生成交互式卡片
  ↓
返回 Lark + Excel
```

**技术亮点**:
- ✅ 多维度数据聚合
- ✅ 智能评级算法
- ✅ 交互式卡片展示
- ✅ 详细报告生成

```

│   用户/系统      │
│   (Lark/Web/n8n)│
└────────┬────────┘
         │
         ↓
┌─────────────────┐

│  │智能查询流程│ │
│  │定时报表流程│ │
│  │评级报告流程│ │
│  └────────────┘ │
└────────┬────────┘
         │ HTTP API
         ↓
┌─────────────────┐
│  Webhook API    │
│  (API Key 认证) │
└────────┬────────┘
         │
         ↓
┌─────────────────┐
│  Athena Service │
│  ┌────────────┐ │
│  │查询优化器  │ │
│  │内存管理器  │ │
│  │结果处理器  │ │
│  └────────────┘ │
└────────┬────────┘
         │
         ↓
┌─────────────────┐
│   AWS Athena    │
│   (数据查询)    │
└─────────────────┘
```

```
查询请求 → API 认证 → 查询优化 → Athena 执行 → 结果处理 → 格式化输出
```
