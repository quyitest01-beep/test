# 将评分数据写入 Google Sheets 配置指南

## 工作流结构

```
计算评分数据 → 整理数据-谷歌表格 → 更新谷歌表格
```

## 节点配置详解

### 1. 计算评分数据节点

这个节点已经配置好，输出结构化的周报数据，包含：
- `platform`: 平台总览数据
- `merchants`: 商户榜单数据
- `games`: 游戏榜单数据
- `currencies`: 币种数据

### 2. 整理数据-谷歌表格节点（Code 节点）

**代码**: 使用 `prepare-google-sheets-data.js` 中的代码

**功能**:
- 将上游的嵌套 JSON 数据转换为扁平化的行数据
- 格式化数值和百分比
- 生成多行数据（平台总览 + 商户榜单 + 游戏榜单 + 币种数据）

**输出格式**:
每行包含以下字段（根据数据类型可能略有不同）：
- `数据类型`: 标识数据类别（平台总览、商户-投注用户Top、游戏-GGRTop 等）
- `排名`: 榜单排名（如适用）
- `商户名称` / `游戏名称` / `币种`: 名称字段
- 各种指标字段（投注用户数、新用户数、留存率、GGR 等）

### 3. 更新谷歌表格节点（Google Sheets 节点）

#### 配置步骤

1. **Operation（操作）**: `Append or Update`

2. **Document ID（文档 ID）**: 
   - 你的 Google Sheets 文档 ID
   - 从 URL 中提取：`https://docs.google.com/spreadsheets/d/{DOCUMENT_ID}/edit`

3. **Sheet Name（工作表名称）**: 
   - 选择要写入的工作表（如 "工作表1"）

4. **Column to Match On（匹配列）**: 
   - 如果使用 `Append or Update`，需要指定用于匹配的列
   - 建议使用 `数据类型` 列（如果希望每次更新时替换旧数据）
   - 或者使用 `报告周期` + `数据类型` 的组合

5. **Columns（列映射）**: 
   - **Mapping Mode**: `Define Below`
   - **列映射配置**:

## Google Sheets 表格结构建议

建议在 Google Sheets 中创建以下列（按顺序）：

### 基础信息列
1. **数据类型** - 标识数据类别
2. **排名** - 榜单排名
3. **报告周期** - 报告时间范围
4. **周期开始** - 周期开始日期
5. **周期结束** - 周期结束日期

### 名称列
6. **商户名称** - 商户名称
7. **游戏名称** - 游戏名称
8. **币种** - 币种代码

### 用户指标列
9. **投注用户数** - 投注用户总数
10. **新用户数** - 新用户数
11. **活跃用户次日留存** - 活跃用户 D1 留存率
12. **活跃用户3日留存** - 活跃用户 D3 留存率
13. **新用户次日留存** - 新用户 D1 留存率
14. **新用户3日留存** - 新用户 D3 留存率

### 营收指标列
15. **总投注USD** - 总投注金额（USD）
16. **总派奖USD** - 总派奖金额（USD）
17. **总GGR_USD** - 总 GGR（USD）
18. **人均GGR_USD** - 人均 GGR（USD）
19. **RTP** - 返奖率

### 游戏指标列
20. **总局数** - 游戏总局数
21. **空转指数** - 空转指数

### 币种指标列
22. **占比** - 币种占比

### 元数据列
23. **生成时间** - 数据生成时间

## 注意事项

1. **列映射**: 
   - 确保 Google Sheets 节点中的列名与"整理数据-谷歌表格"节点输出的字段名完全一致
   - 如果某些字段在某些行中不存在，n8n 会自动处理为空值

2. **数据类型**:
   - 建议在 Google Sheets 中设置正确的数据类型（数字、百分比、日期等）
   - 可以在 Google Sheets 中手动格式化，或使用公式自动格式化

3. **性能优化**:
   - 如果数据量很大，考虑分批写入
   - 可以使用 `append` 模式避免每次更新时查询现有数据

4. **错误处理**:
   - 确保 Google Sheets API 凭证有足够的权限
   - 检查工作表名称是否正确
   - 确认列映射配置正确

## 验证步骤

1. **检查上游数据**:
   - 在"整理数据-谷歌表格"节点的 OUTPUT 中查看数据格式
   - 确认所有字段都正确输出

2. **测试写入**:
   - 执行 Google Sheets 节点
   - 检查 OUTPUT 是否返回成功信息

3. **验证表格**:
   - 在 Google Sheets 中查看数据是否正确写入
   - 检查数据类型和格式是否正确
