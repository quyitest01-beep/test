## 五、数据过滤规则

### Retention数据过滤
- **只保留 `dailyUsers >= 50` 的记录**
- 如果原始数据中 `dailyUsers < 50`，该记录会被过滤掉

### 留存用户数计算
- **公式**: `retainedUsers = Math.round(dailyUsers * (retention / 100))`
- **示例**: 
  - dailyUsers = 191, retention = 20.94%
  - retainedUsers = Math.round(191 * 0.2094) = 40
