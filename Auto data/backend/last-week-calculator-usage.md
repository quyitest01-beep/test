# 上周日期计算器使用指南

## 📋 概述

这个Code节点用于根据今日日期计算上周的日期范围，输出格式为 `YYYYMMDD-YYYYMMDD`。

## 🎯 功能特点

- **自动计算上周日期范围**：根据今日日期自动计算上周一到上周日的日期范围
- **支持多种日期格式**：支持 `2025.10.21`、`2025-10-21`、`20251021` 等格式
- **智能星期计算**：正确处理一周内不同日期的上周计算
- **标准输出格式**：输出 `YYYYMMDD-YYYYMMDD` 格式的日期范围

## 🔧 使用方法

### 1. 在n8n中使用

1. 添加一个 **Code** 节点
2. 将 `final-last-week-calculator.js` 的代码复制到Code节点中
3. 运行节点

### 2. 输入参数（可选）

```javascript
// 可以通过输入数据提供今日日期
{
  "todayDate": "2025.10.21",  // 可选，不提供则使用当前日期
  "date": "2025.10.21",       // 可选，同todayDate
  "today": "2025.10.21"       // 可选，同todayDate
}
```

### 3. 输出结果

```javascript
{
  "lastWeekRange": "20251013-20251019",  // 上周日期范围
  "lastWeekStart": "20251013",           // 上周一
  "lastWeekEnd": "20251019",             // 上周日
  "today": "2025-10-21",                 // 今日日期
  "todayDayOfWeek": 2,                   // 今日星期几 (0=周日, 1=周一, ..., 6=周六)
  "todayDayOfWeekName": "周二"            // 今日星期几名称
}
```

## 📊 计算逻辑

### 算法说明

1. **获取今日日期**：从输入参数或当前系统时间获取
2. **计算今日星期几**：使用 `getDay()` 方法获取 (0=周日, 1=周一, ..., 6=周六)
3. **计算上周一**：
   - 如果今日是周日(0)，上周一是6天前
   - 如果今日是周一到周六(1-6)，上周一是 (今日星期几 + 6) 天前
4. **计算上周日**：上周一 + 6天
5. **格式化输出**：转换为 `YYYYMMDD` 格式

## 🚀 实际应用场景

### 1. 在商户分析工作流中使用

```javascript
// 在n8n工作流中，可以这样使用：
// 1. 获取上周日期范围
const lastWeekRange = $json.lastWeekRange; // "20251013-20251019"

// 2. 分割日期范围
const [startDate, endDate] = lastWeekRange.split('-');
// startDate = "20251013", endDate = "20251019"

// 3. 用于Athena查询
const sqlQuery = `
SELECT 
    date_str,
    merchant,
    unique_users
FROM merchant_game_analytics
WHERE stat_type = 'merchant_daily'
    AND date_str >= '${startDate}'
    AND date_str <= '${endDate}'
ORDER BY date_str, merchant;
`;
```

### 2. 在Telegram Bot中使用

```javascript
// 发送上周统计报告
const message = `📊 **上周商户统计报告** (${lastWeekRange})
• 统计时间: ${lastWeekStart} 至 ${lastWeekEnd}
• 今日: ${today} (${todayDayOfWeekName})
• 数据已更新，请查看详细报告`;
```

### 3. 在数据导出中使用

```javascript
// 生成文件名
const filename = `merchant_analytics_${lastWeekRange}.csv`;

// 设置查询参数
const queryParams = {
  startDate: lastWeekStart,
  endDate: lastWeekEnd,
  statType: 'merchant_daily'
};
```
