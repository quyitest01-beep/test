# 商户数据处理器使用指南

## 功能概述

这个商户数据处理器能够智能识别上游数据结构，并生成相应的Lark表格格式数据。

## 支持的数据结构

### 1. 商户映射数据
**上游数据结构**：
```javascript
{
  "status": "success",
  "timestamp": "2025-10-22T05:34:29.534Z",
  "statistics": {
    "total_rows": 1998,
    "processed_rows": 201,
    "skipped_rows": 1797,
    "production_merchants": 201,
    "success_rate": "10.1%"
  },
  "filtered_merchants": [
    {
      "sub_merchant_name": "betfiery",
      "main_merchant_name": "RD1",
      "merchant_id": 1698202251
    },
    // ... 更多商户数据
  ]
}
```

**输出格式**：
```javascript
{
  "商户名": "betfiery",
  "主商户名": "RD1",
  "商户ID": 1698202251
}
```

### 2. 商户活跃用户数据
**上游数据结构**：
```javascript
{
  "merchantData": [
    {
      "json": {
        "merchant": "12POKIES",
        "stat_type": "merchant_daily",
        "daily_unique_users": "1525",
        "date_str": "20251013",
        "isMatched": true
      }
    }
    // ... 更多数据
  ]
}
```

**输出格式**：
```javascript
{
  "商户名": "12POKIES",
  "日期": "合计",
  "投注用户数": 1525
}
```

```javascript
{
  "商户名": "12POKIES",
  "日期": "20251013",
  "投注用户数": 1525
}
```

## 处理逻辑

### 1. 数据结构识别
- 检查是否有 `filtered_merchants` 字段
- 检查是否有 `merchantData` 字段
- 根据数据结构选择相应的处理方式

### 2. 数据排序
- **商户名排序**：按A→Z字母顺序排序
- **日期排序**：每个商户内的日期按时间顺序排序

### 3. 数据生成
- **商户映射数据**：直接输出商户信息
- **活跃用户数据**：生成合计行和每日数据

## 使用场景

### 场景1：商户映射数据
- 输入：包含 `filtered_merchants` 的数据
- 输出：按商户名排序的商户映射信息

### 场景2：商户活跃用户数据
- 输入：包含 `merchantData` 的数据
- 输出：按商户名排序的活跃用户统计

## 特性

1. **智能识别**：自动识别上游数据结构
2. **灵活处理**：支持多种数据格式
3. **排序功能**：支持中英文混合排序
4. **格式统一**：输出标准的Lark表格格式

## 注意事项

1. 确保上游数据包含相应的字段
2. 商户名必须是字符串类型
3. 用户数字段会自动转换为数字类型
4. 日期格式为 YYYYMMDD 字符串

## 调试信息

处理器会输出详细的调试信息：
- 数据结构分析
- 字段识别结果
- 数据统计信息
- 排序结果示例
