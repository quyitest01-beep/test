# AI 场景匹配问题分析与修复建议

## 问题分析

**用户查询**："查一下商户1740973068的2025.11月的投注记录"

**应该匹配的场景**：S3（时间范围全量查询）

**实际可能匹配的场景**：无或错误场景

## 根本原因

1. **S1场景定义**：S1是"单/多记录 ID 查询"，必填字段是`id`（记录ID）
   - 用户查询中没有提供任何ID，所以**不应该匹配S1**
   
2. **应该匹配S3的原因**：
   - S3场景：时间范围全量查询
   - 必填字段：`start_date`, `end_date`
   - 可选字段：`merchant_id`, `game_code`, `provider`
   - 用户查询包含：商户ID（1740973068）+ 时间范围（2025.11月）+ 查询对象（投注记录）

3. **可能的问题**：
   - "获取知识库场景"工具可能没有正确返回S3场景
   - AI没有正确理解工具返回的场景列表
   - 提示词中的场景匹配逻辑不够明确

## 修复建议

### 1. 优化提示词中的场景匹配逻辑

在提示词的"场景识别规则"部分，添加更明确的场景匹配优先级：

```
**场景匹配优先级**：
1. 如果查询包含 `id`（记录ID，纯数字，长度≥16）→ 匹配 S1
2. 如果查询包含 `uid`（用户ID）且无 `id` → 匹配 S2
3. 如果查询包含 `merchant_id` + `时间范围` + "投注记录"/"明细" → 匹配 S3
4. 如果查询包含 `merchant_id` + `时间范围` + "统计"/"汇总" → 匹配 S4
5. 如果查询包含 `game_code` + `时间范围` → 匹配 S5
6. 其他情况根据工具返回的场景列表匹配
```

### 2. 增强场景匹配说明

在提示词中添加明确的场景匹配示例：

```
**场景匹配示例**：
- "查一下商户1740973068的2025.11月的投注记录" → S3（时间范围全量查询）
  - 原因：包含merchant_id、时间范围、查询对象是"投注记录"（明细）
- "查一下商户1740973068的2025.11月的统计数据" → S4（商户维度全量统计）
  - 原因：包含merchant_id、时间范围、查询对象是"统计"（聚合）
- "查一下ID 1990823099625480192" → S1（单/多记录ID查询）
  - 原因：包含id字段
```

### 3. 优化工具调用说明

在提示词中明确说明如何理解工具返回的场景：

```
**工具返回场景的理解**：
1. 工具返回的场景列表是权威来源，必须优先使用
2. 如果工具返回多个候选场景，选择最匹配的一个（根据必填字段判断）
3. 如果工具返回为空，根据"查数三要素"和场景定义自主判断
4. 在reason中必须说明匹配依据，引用工具返回的场景信息
```

### 4. 修改后的提示词关键部分

```markdown
**场景识别规则（重要）**：
1. **必须调用"获取知识库场景"工具**，传入完整查数需求（包括merchant_id、时间范围、查询对象等）
2. **工具返回的场景列表是唯一权威来源**，必须根据工具返回的场景定义进行匹配
3. **场景匹配优先级**（当工具返回多个候选时）：
   - 如果查询包含 `id`（记录ID）→ 优先 S1
   - 如果查询包含 `merchant_id` + 时间范围 + "投注记录"/"明细" → 优先 S3
   - 如果查询包含 `merchant_id` + 时间范围 + "统计"/"汇总" → 优先 S4
   - 如果查询包含 `game_code` + 时间范围 → 优先 S5
4. **在reason中必须说明**：
   - 工具返回了哪些场景
   - 为什么选择这个场景（引用场景的必填字段、可选字段）
   - 如果工具返回为空，说明原因并自主判断
```

## 验证方法

1. **检查工具返回**：确认"获取知识库场景"工具是否正确返回了S3场景
2. **检查AI输出**：查看AI的reason字段，是否说明了匹配依据
3. **测试用例**：
   - "查一下商户1740973068的2025.11月的投注记录" → 应该匹配S3
   - "查一下ID 1990823099625480192" → 应该匹配S1
   - "查一下用户li-xxx的投注记录" → 应该匹配S2

## 总结

**核心问题**：用户的查询"商户+时间范围+投注记录"应该匹配S3场景，而不是S1场景。S1场景要求必填字段是`id`（记录ID），而用户查询中没有ID。

**解决方案**：
1. 优化提示词，明确场景匹配优先级和示例
2. 确保工具正确返回场景列表
3. 增强AI对工具返回结果的理解能力
